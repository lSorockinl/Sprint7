<h2 id="technologies">Использованные технологии</h2>

В проекте были использованы следующие фреймворки/библиотеки:
* Java 11,
* JUnit 4.13.2,
* RestAssured 5.3.0,
* Allure-junit4 2.22.0,
* GSON 2.10.1

<!-- Генерирование отчета -->
<h2 id="start-tests">Генерирование отчета</h2>

* mvn
  ```sh
  mvn clean test
  ```
* mvn
  ```sh
  mvn allure:serve
  ```
